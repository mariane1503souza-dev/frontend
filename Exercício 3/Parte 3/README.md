# Exercício 3 - Front-end
Este repositório contém a resolução final do Exercício 3 (Partes 1, 2 e 3), focando em identidade visual e estilização avançada.
## Estrutura do Projeto
index.html: Estrutura principal da landing page.
css/style.css: Estilos externos aplicados ao projeto.
README.md: Documentação básica.
## Requisitos Implementados (Completo)
- Identidade Visual: Paleta de cores definida (Verde, Amarelo, Off-white e Azul Escuro).
- Estilização de Componentes:
- Cards: Com background-color, border-radius e box-shadow.
- Botão Principal: Com cor forte, texto legível e efeitos de hover.
- Melhoria de Espaçamentos: Uso estratégico de padding e margin para conforto visual.
- Organização por Seletores: Uso de IDs para seções e classes reutilizáveis para elementos comuns.
- Navegação Funcional: Menu com âncoras para navegação na mesma página.
- Boas Práticas: 100% do estilo em arquivo externo, sem CSS inline.
