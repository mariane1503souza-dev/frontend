# Exercício 3 - Front-end
Este repositório contém a resolução do Exercício 3 (Partes 1 e 2), conforme solicitado.
## Estrutura do Projeto
- index.html: Estrutura principal da landing page.
- css/style.css: Estilos externos aplicados ao projeto.
## README.md: Documentação básica.
### Requisitos Implementados (Parte 1 & 2)
 - Melhoria na Navegação: Links configurados para #inicio, #sobre, #beneficios e #contato.
 - Uso de IDs: Cada seção possui um ID único correspondente ao menu.
### Classes Reutilizáveis:
.container: Para limitar a largura do conteúdo.

.botao-principal: Para o botão de chamada (CTA).

.card: Para os blocos de benefícios/recursos.

.secao: Para espaçamento vertical padronizado.

 **Estilização Externa**: Uso exclusivo de css/style.css com seletores de tag, classe e ID.
 Sem CSS Inline: Nenhuma tag style ou atributo style foi utilizado.
