## Como corrigir uma imagem quebrada?
Uma imagem aparece "quebrada" no navegador quando o caminho especificado no atributo 'src' da tag '<img>' não aponta para um arquivo válido ou existente. para corrigir isso, siga estes passos:

1. **Verifique o Caminho (Path):**
 Certifique-se de que o caminho está correto em relação a localização do arquivo HTML. Se a imagem está em uma pasta chamada 'img/nome-da-imagem.jpg'.
 2. **Confira a Extensão:**
 Verifique se a extensão do arquivo está correta (ex: '.png', '.jpg'). Lembre-se que o GitHub diferencia maiúsculas de minúsculas.
 3. **Verifique a Existência do Arquivo:**
Confirme se o arquivo da imagem foi realmente salvo na pasta 'img/' e se o nome não possui espaços
 4. **Atributo Alt:**
Sempre use o atributo 'alt' para acessibilidade e para que um texto apareça caso a imagem falhe.
